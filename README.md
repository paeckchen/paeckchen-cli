CLI interface of [paeckchen](https://github.com/paeckchen/paeckchen).

## Installation

Grab it from npm

```shell
npm install paeckchen-cli
```

## Usage

```js
import { paeckchenCli } from 'paeckchen-cli';
paeckchenCli();
```
